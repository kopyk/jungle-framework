# Jungle Decision Framework

A decision-driven macro framework for **jungling in League of Legends**.

This project is styled as “documentation-first” and focuses on **repeatable decisions**:

## What this is
- A learning + example documentation project
- A scalable macro system from Iron → Challenger
- Decision rules, decision trees (early + mid), examples, and a glossary


## Local development
```bash
npm install
npm start


```

This website is built using [Docusaurus](https://docusaurus.io/), a modern static website generator.
